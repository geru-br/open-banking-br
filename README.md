# open-banking-br

## Consultas:

### Saldo e Limites

```
  /accounts/{AccountId}/balances
  /balances:
```

### Extratos

```
  /accounts/{AccountId}/transactions
  /transactions
```

### Dados da conta (Conta simples)

```
  /accounts/{AccountId}/party
  /party
```

### Dados da conta (Conta conjunta e juridica)

```
  /accounts/{AccountId}/parties
```
